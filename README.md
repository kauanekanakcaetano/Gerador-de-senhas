  Gerador de Senhas

##  Sobre o Projeto

O **Gerador de Senhas** é uma aplicação web desenvolvida com **HTML5**, **CSS3** e **JavaScript**. Seu objetivo é gerar senhas fortes e seguras de forma rápida, permitindo ao usuário personalizar a quantidade de caracteres e os tipos de caracteres utilizados.

---

##  Funcionalidades

-  Gerar senhas aleatórias.
-  Definir a quantidade de caracteres.
-  Incluir letras maiúsculas.
-  Incluir letras minúsculas.
-  Incluir números.
-  Copiar a senha gerada com um clique.
-  Exibir a força da senha.
-  Mostrar uma estimativa do tempo para descobrir a senha.

---

##  Como Executar

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo **index.html** em qualquer navegador.
4. Escolha as configurações desejadas.
5. Clique em **✨  Gerar Senha**.
